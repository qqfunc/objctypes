# Development Guide for Gemini

# Development Commands

- Install dependencies with `uv run sync`.
- Install Git Hooks with `uv run lefthook install`.

## Coding Conventions

### Python

Follow PEP 8.

- Indents: 4 spaces
- String Literals: double quotes

### C / C++ / Objective-C

Follow PEP 7.

- Indents: 4 spaces
- String Literals: double quotes

## Commit Messages

Follow Conventional Commits.
